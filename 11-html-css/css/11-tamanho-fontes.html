Tamanho de fontes

medidas absolutas

  • cm - a apresentação pode variar pra cada tela
  • mm - a apresentação pode variar pra cada tela
  • in - polegada
  • px - pixel (recomendável para tamanho de fonte)
  • pt - medida de tipografia em papel, é a fonte utilizada no word (Não recomendável)
  • pc - medida de tipografia em papel (Não recomendável)

Medidas relativas

  • em - altura relativa ao "m" maiúsculo de uma fonte (recomendável para tamanho de fonte)
  • ex - altura relativa a altura "x" de uma fonte
  • vw - viewport width - relativo ao tamanho da tela
  • vh - viewport height - relativo ao tamanho da tela
  • %

Boas práticas

  • Normalmente é usado 16px para tamanho de fontes
  • 16px geralmente é o tamanho de 1em
